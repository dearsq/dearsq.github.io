MarkDown语法学习

发表于 2016-01-04   |  

更详细的 Markdown 语法参考 http://www.appinn.com/markdown/basic.html

一、What is Markdown

Markdown 是一种用来写作的轻量级「标记语言」。我们采用标记语法，来代替常见的排版格式。并且可以导出为多种格式。

二、Markdown’s grammar

标题

如果一段文字被定义为标题，只需要在这段文字前面加 # 号即可。

总共六级标题，# 号后需要空格。

列表

列表的显示只要在文字前加上 - 或者 * 即可变成无序，有序列表直接在文字前加上 1. 2. 3. 符号，符号和文字之间需要加上一个字符的空格。

无序列表

  • 第一条
  • 第二条
  • 第三条

有序列表

  1. 第一条
  2. 第二条
  3. 第三条

引用

如果需要引用一小段别处的句子，就需要引用的格式。需要在文本前加入 > 这种大于号即可。

这里是引用

图片和链接

图片的语法为：{ImgCap}{/ImgCap}

链接的语法为：

Baidu

粗体与斜体

用两个 “*” 包含一段文本就是粗体的语法，用一个 “” 包含一段文本的就是斜体的用法

这里是粗体 这里是斜体

表格

表格很麻烦，不推荐使用表格。

代码引用

需要使用两个`将要引用的代码包裹起来。

分割线

分割线只需要三个 * 来实现
如
#MarkDown Learning
